# Squiggl - Draw and guess game

Demo: https://squiggl.herokuapp.com/

### Goals
Learn React and realtime communication with socket.io

### Features
- Realtime drawing and chat
- Multi round games
- Turn based drawing
- Players able to join while game is in session
- Score keeping and leaderboard
